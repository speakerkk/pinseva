How a PIN Code Works

A Postal Index Number (PIN) is a 6-digit numeric code used by India Post. Each digit has a specific meaning:

  • First Digit: Indicates the postal circle region.
  • First 2 Digits: Indicates the postal circle or sub-region.
  • First 3 Digits: Indicates the specific sorting or revenue district.
  • Last 3 Digits: Indicates the specific delivery Post Office.
